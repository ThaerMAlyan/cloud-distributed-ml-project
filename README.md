# Cloud and Distributed Systems Project

This project is developed as part of the **Cloud and Distributed Systems** course.

## Project Overview
A cloud-based data processing and machine learning system implemented using **Apache Spark (PySpark)** on **Databricks**.  
The project demonstrates distributed data processing, machine learning, and scalability analysis.

## Features
- Data preprocessing and statistical analysis
- Machine Learning using Spark MLlib:
  - Linear Regression
  - KMeans Clustering
  - Logistic Regression (Classification)
  - FP-Growth
- Performance benchmarking
- Scalability analysis using increasing data sizes

## Technologies Used
- Apache Spark (PySpark)
- Databricks
- Python
- GitHub
